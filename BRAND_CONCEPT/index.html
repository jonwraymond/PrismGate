    PrismGate Brand Concept¶

    Logo Concept¶

    Primary Logo: Prism Refraction¶

    A geometric prism (triangle) refracting a single beam of light into multiple colored rays representing different MCP tools/connections. The prism is the gateway; the rays are the tools it discovers and routes.

            ┌─────────────────┐
        │                 │
        │    ╱╲           │
        │   ╱  ╲          │
        │  ╱ ◇  ╲         │
        │ ╱      ╲        │
        │╱────────╲       │
        │  ╱││││╲         │
        │ ╱ ││││ ╲        │
        │╱  ││││  ╲       │
        │   ││││   ╲      │
        └─────────────────┘
           ││││
           ││││ → Tool rays (cyan, violet, emerald, amber)
           ││││

    Variations¶

    • Icon only — Prism with 3-4 rays (for favicons, app icons)
    • Wordmark — "PrismGate" in Inter Bold (for headers, docs)
    • Lockup — Icon + wordmark (for marketing materials)

    Logo Usage Rules¶

    • Minimum size: 32px height
    • Clear space: 1x prism width on all sides
    • Dark backgrounds: Use light prism with colored rays
    • Light backgrounds: Use dark prism with colored rays

    Color Palette¶

    Primary Colors¶

    Color Hex Usage
    Deep Violet #6B21A8 Primary brand color, headers, CTAs
    Cyan #06B6D4 Accent, links, highlights
    Slate #1E293B Dark backgrounds, text

    Secondary Colors¶

    Color Hex Usage
    Emerald #10B981 Success states, test pass
    Amber #F59E0B Warning states, attention
    Rose #F43F5E Error states, failures
    Violet Light #A78BFA Lighter accents, hover states

    Ray Colors (Logo)¶

    Ray Color Represents
    Ray 1 Cyan #06B6D4 Tool discovery
    Ray 2 Emerald #10B981 Session isolation
    Ray 3 Amber #F59E0B Secrets management
    Ray 4 Rose #F43F5E Health monitoring

    • Code: JetBrains Mono (400, 500)

    Brand Voice¶

    Personality Traits¶

    • Technical but accessible — Explain complex concepts clearly
    • Confident — State capabilities without hedging
    • Developer-focused — Speak to developers, not managers
    • Transparent — Open about limitations and trade-offs

    Tone Examples¶

    Context Tone Example
    README Helpful "Getting started takes 3 commands."
    Error messages Direct "Backend 'github' failed health check. Retrying in 5s."
    Marketing Confident "Intelligent tool discovery. Isolated sessions. Secure secrets."
    Docs Precise "The search_tools endpoint accepts a task_description string and returns up to limit results."

    Words to Use¶

    • intelligent, discover, isolate, secure, performant
    • gateway, prism, refactor, compose, orchestrate
    • transparent, open-source, vendor-free

    Words to Avoid¶

    • revolutionary, game-changing, next-gen
    • simple, easy (implies triviality)
    • just, merely (minimizes complexity)

    Visual Style Guide¶

    Documentation Style¶

    • Dark theme primary (#1E293B background, white text)
    • Code blocks: JetBrains Mono, #0F172A background
    • Syntax highlighting: Violet for keywords, Cyan for strings, Emerald for functions
    • Diagrams: Clean lines, minimal color, consistent spacing

    Marketing Materials¶

    • Hero images: Dark gradient background (violet → slate)
    • Feature cards: White background, subtle violet border
    • Screenshots: Dark theme UI with cyan accent highlights
    • Social media: 16:9 aspect ratio, logo in top-left, tagline centered

    Iconography¶

    • Line icons (1.5px stroke)
    • Rounded corners (2px radius)
    • Consistent with Lucide icon set style
    • Custom icons for: prism, tool discovery, session isolation, secrets
